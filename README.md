# Advanced OSINT Reverse Lookup Tool

An ethical Python tool for reverse phone/name/email investigations, inspired by agency workflows.

## Features
- Multi-source scraping (NumLookup, HaveIBeenPwned).
- Google dorks for social enumeration.
- Breach detection and entity graphs.

## Setup
1. Clone: `git clone https://github.com/yourusername/Advanced-OSINT-Reverse-Lookup-Tool.git`
2. Install: `pip install -r requirements.txt`
3. Run: `python advanced_osint_tool.py`
